# MenuLens Android Visual System

MenuLens uses a minimalist Japanese editorial direction: near-white paper,
charcoal text, restrained vermilion actions, moss secondary accents, muted
gold utility details, serif display typography, and compact sans-serif body copy.

## Principles

- Dish information is the visual subject; decoration stays quiet.
- Display headings use serif typography. Controls and explanatory copy use sans serif.
- Cards use fine hairline borders, very low elevation, and small corner radii.
- Decoration is removed unless it carries meaning; photography, menu names, and spacing create the visual rhythm.
- Locked content never renders English text, tags, or identifying imagery.

## Screen behavior

- Scan: editorial Japanese storefront hero, full-width camera/upload actions, and icon-led photography guidance.
- Processing: OCR, dish separation, and English guidance stages only. Image generation is excluded.
- Results: compact menu rows with Japanese name, price, lock/media tile, and chevron. English and cached thumbnails appear only after reveal.
- Detail: compact app bar, Japanese identity first, 4:3 generated-image hero, caption, explanation, tags, disclaimer, and persistent Show-to-Staff action.
- Show to Staff: only 「これをください」, the Japanese dish name, and optional price.
- Paywall: benefits, remaining daily reveals, restore action, and clearly labeled debug controls.

## Launcher icon

The launcher icon combines a vermilion camera lens with a vertical menu slip.
It uses the same paper, charcoal, vermilion, and muted-gold palette as the app
and keeps generous safe-zone padding for Android launcher masks.
